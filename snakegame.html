Snake Game with Hand Tracking

  • Tech Stack: Python, OpenCV, cvzone, NumPy, HandTrackingModule
  • Features: Real-time Hand Tracking, Gesture-Based Control, Collision Detection, Score System
  • Libraries Used: OpenCV, cvzone, math, random, NumPy
  • Github URL: Project Link
  • Live Demo: Click for Demo

This project is a modern twist on the classic Snake Game, controlled entirely by hand gestures using a webcam. It uses the cvzone Hand Tracking Module to detect and track the player’s index finger, guiding the snake’s movement in real time. The game includes a scoring system, food spawning, snake growth, and collision detection to end the game when the snake hits itself. Players can restart anytime by pressing R, and the game runs smoothly with live OpenCV rendering. This is an interactive computer vision project demonstrating gesture-based control in games.
