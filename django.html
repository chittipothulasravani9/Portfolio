Django CRUD Operation

  • Tech Stack: Python, Django, HTML, CSS, Bootstrap, PostgreSQL
  • Database: PostgreSQL
  • Github URL: Project Link
  • Live Demo: Click for Demo

This project is a complete implementation of CRUD (Create, Read, Update, Delete) operations using Django. It includes form handling, database integration with PostgreSQL, and responsive UI built with Bootstrap. Users can add, edit, delete, and view records in a clean, user-friendly interface. Authentication and validation are also implemented to ensure secure and accurate data handling.
